Current Prototype (Proof of Concept)

ESP32-C3 SuperMini

Runs lightweight edge AI (sqrt(ax²+ay²+az²)) to filter vibrations. Sends JSON events to dashboard via Wi-Fi.

GY-521 MPU6050

I2C accelerometer/gyroscope. Detects low (≥1.5g), medium (≥2.0g), and high (≥3.0g) severity impacts.

*Note: The current prototype only proves vibration-based detection and dashboard communication. It does NOT include GPS, dashcam, AI card, or image processing.*
Fleet Deployment Architecture (Production)

Low-Cost Vibration Trigger PCB

ESP32 + MPU6050 + 12V/24V to 5V Buck Converter + Fuse tap. Powered safely from the vehicle fuse box.

Reuse Existing Fleet Dashcams

ESP32 sends a GPIO/UART trigger to the fleet's existing telematics unit to capture 1 snapshot + GPS + Timestamp.

Depot Wi-Fi Upload

Events are stored locally on the dashcam and uploaded when the bus returns to base. No LTE cost, no heavy video uploads.

Software Stack

Firmware: Arduino C++ (Edge detection)
Backend: Node.js, Express, WebSocket (Render)
Frontend: HTML/CSS/JS, Leaflet maps, ApexCharts

"In deployment, Road Sense reuses existing fleet dashcams and GPS systems. We only add a low-cost ESP32 vibration trigger PCB powered from the fuse box. When abnormal impact is detected, it triggers the dashcam to capture one snapshot, attaches GPS, stores it locally, and uploads via depot Wi-Fi. This avoids constant internet dependency and keeps hardware costs very low."

Challenges & Risks

🖼️ Visuals: Two-column layout: Challenge on left, Mitigation on right.
  • Challenge: Hardware durability (heat, vibration, dust) and power.
    → Mitigation: Ruggedized enclosure, and using a 12V/24V to 5V buck converter safely tapped from the vehicle fuse box.
  • Challenge: False positives — speed bumps or rough surfaces looking like potholes.
    → Mitigation: Edge waveform analysis and cooldown timers. The dashcam snapshot allows operators to visually verify defects before dispatch.
  • Challenge: GPS and Data Costs.
    → Mitigation: Reusing the fleet's existing telematics unit for GPS tagging and uploading via Depot Wi-Fi entirely eliminates LTE costs.
"Our main challenges are false positives and hardware durability. We mitigate false positives with edge analysis and visual confirmation from the fleet's existing dashcam. And by reusing the fleet's GPS and uploading via Depot Wi-Fi, we completely eliminate cellular data costs."
